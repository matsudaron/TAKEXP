■ サービス概要

TAKEXPは、日々のタスク継続を「ゲーム化」して管理できる自己成長ツールです。
タスクをこなすことで経験値（XP）が溜まり、進捗が視覚的に積み上がることで、継続のモチベーションを維持します。

⸻

■ このサービスへの思い・作りたい理由

「継続できない」の原因は意志の弱さではなく、成果の可視化不足だと考えています。

やったことが積み上がっている実感がないと、人は続けられない。
逆に、小さくても“進んでいる感覚”があれば継続は自然に起きる。

TAKEXPは、その“継続の可視化”をゲーム的なUXで解決するために作っています。

⸻

■ ユーザー層について

* 自己管理が苦手な人
* 習慣化に失敗しがちな人
* 勉強・筋トレ・副業などを継続したい人
* 目標達成をゲーム感覚で楽しみたい人

⸻

■ サービスの利用イメージ

1. タスクを登録する
2. 毎日「+1 day」で進捗を積み上げる
3. XPバーが増えていく
4. 一定達成で演出が発生
5. 継続が“見える形”で残る

⸻


■ サービスの差別化ポイント・推しポイント

* タスク管理 × ゲーム化（XP・進捗バー）
* シンプルなUIで継続ストレスを排除
* 達成時の演出によるモチベーション設計
* Supabaseによる軽量かつ即時同期型構成
* “努力の可視化”に特化した設計思想

⸻

■ 機能候補

・アカウント実装

* Supabase Authによるログイン・新規登録
* プロフィール管理（username）

・タスク管理

* タスク作成 / 編集 / 削除
* goal設定（目標日数）
* 進捗（days）管理

・ゲーム要素

* XPバー表示
* +1 dayで進捗加算
* 達成時の演出（celebration）

・プロフィール機能

* ニックネーム変更
* タスクデータ削除

⸻

■ 機能の実装方針予定

* フロント中心（React SPA）
* 状態管理はuseStateベースで軽量設計
* DBはSupabase（Postgres）でリアルタイム同期
* 認証はSupabase Authに集約

⸻

■ 使用予定の技術スタック

* Frontend: React (JavaScript / TypeScript)
* Backend: Supabase (PostgreSQL + Auth)
* Hosting: 未定
* VCS: GitHub
* 開発環境: Docker
